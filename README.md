# Phishing Email Detection System

A Machine Learning based cybersecurity project that detects phishing emails.

## Features
- Email classification
- ML-based detection
- Web interface using Flask
- Real-time prediction

## Technologies Used
- Python
- Flask
- Scikit-learn
- HTML/CSS

## Project Structure
app.py
train_model.py
detector.py

model.pkl
vectorizer.pkl
templates/index.html
