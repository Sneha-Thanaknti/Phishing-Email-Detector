# Phishing Email Detector

A Machine Learning based phishing email detection system built using Python, Flask, Scikit-Learn, SQLite, and a real-world email dataset.

## Features

* Detects phishing and safe emails
* Confidence score prediction
* Scan history storage
* SQLite database integration
* Dashboard analytics
* CSV report download
* Machine Learning classification using TF-IDF and Logistic Regression

## Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* SQLite
* HTML/CSS

## How to Run

1. Install dependencies
2. Train the model:

   ```
   python train_model.py
   ```
3. Start the application:

   ```
   python app.py
   ```
4. Open:

   ```
   http://127.0.0.1:5000
   ```
