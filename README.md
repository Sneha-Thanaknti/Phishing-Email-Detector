# 🛡️ Phishing Email Detection System

A Machine Learning-based cybersecurity project that detects phishing emails and classifies them as **Phishing** or **Safe** using Python, Flask, Scikit-Learn, SQLite, and Matplotlib.

## 🚀 Features

* Phishing Email Detection
* Machine Learning Classification
* Confidence Score Prediction
* Flask Web Dashboard
* SQLite Database Integration
* Scan History Tracking
* Statistics Dashboard
* Pie Chart Visualization

## 🛠 Technologies Used

* Python
* Flask
* Scikit-Learn
* Pandas
* SQLite
* Matplotlib
* HTML/CSS

## ▶️ How to Run

1. Install dependencies

pip install flask pandas scikit-learn matplotlib joblib

2. Train the model

python train_model.py

3. Create database

python database.py

4. Run application

python app.py

5. Open browser

http://127.0.0.1:5000
